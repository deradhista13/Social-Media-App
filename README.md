## Getting Started

1. Clone repo:
```sh
   git clone https://github.com/aryoyudhanto/group1-socialmedia-app
   ```
2. Install package
```sh
   npm install
   ```
3. Run development server
```sh
   npm run dev
   ```

Open [http://localhost:5173/](http://localhost:5173/) with your browser to see the result.
